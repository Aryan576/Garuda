History of C Language

C is the most widely used computer language.

The C programming language is used to program desktop applications, compilers, tools and utilities and even hardware devices.

The C programming language follows a procedure oriented paradigm.

So, In C language, a large program is divided into small programs called functions.

Prime focus is on functions and procedures that operate on the data.

C Program structure follows “Top Down Approach”.
